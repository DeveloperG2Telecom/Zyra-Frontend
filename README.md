# 🏗 Zyra Frontend

Sistema de Documentação e Monitoramento de Equipamentos para Provedora de Internet.

## 🚀 Início Rápido

```bash
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm start

# Acessar aplicação
http://localhost:10000
```
